# Git 使用

## 命令

```bash
# git branch                          列出所有分支
# git branch name                     创建name分支
# git checkout name                   切换到name分支
# git branch -d | -D name             删除name分支
# git branch -d -r name               删除远程name分支
# git g merge Name                    把name分支合并到当前分支上
```

## 开发规范

```bash
# master             主分支
# develop            开发分支
# develop/feature    开发新功能时，以develop为基础创建feature分支,feature-test
# hotfix 分支        修复bug分支
```
